# Project Summary
The project is a full-stack application designed as an AI-driven sales assistant for dietary supplements. It leverages OpenAI's API to provide personalized product recommendations based on users' health goals and preferences through an interactive conversational interface. The aim is to enhance user engagement, facilitate informed purchasing decisions, and improve the overall shopping experience in the dietary products market.

# Project Module Description
The project consists of the following functional modules:
- **User Interaction**: Engages users to collect information about their health needs and preferences.
- **Product Recommendation**: Suggests dietary supplements based on user inputs.
- **Chat Interface**: Provides a user-friendly interface for interaction with the AI assistant.
- **Data Management**: Manages client and product data, integrating new supplements and client profiles.
- **AI Enhancement**: Implements dynamic response generation and contextual memory to improve user interaction, addressing issues such as context retention during conversations.

# File Description Inventory
- **index.js**: Main server file for the backend, setting up the Express server.
- **package.json**: Lists project dependencies and scripts.
- **.env.example**: Template for environment variables.
- **index.html**: Entry point for the React application.
- **ProductCardInChat.tsx**: Component for displaying product information in chat.
- **ChatInterface.tsx**: Component for managing chat interactions.
- **useChat.ts**: Custom hook for handling chat logic and state, including user interactions and AI response generation.
- **clients.ts**: Contains client data and profiles.
- **supplements.ts**: Contains product data for dietary supplements.
- **aiAgent.ts**: Logic for interacting with OpenAI's API and processing user input, including dynamic greeting and context management.
- **openai.ts**: Configuration settings for OpenAI API integration.

# Technology Stack
- **Frontend**: React, TypeScript
- **Backend**: Node.js, Express
- **Database**: In-memory data structures for client and product information
- **AI Integration**: OpenAI API for natural language processing

# Usage
To set up the project, follow these steps:
1. Install dependencies:
   ```bash
   npm install
   ```
2. Create a `.env` file based on `.env.example` and add your OpenAI API key.
3. Start the server:
   ```bash
   npm start
   ```
4. Open the application in your browser.
