# Демо-чат с ИИ-агентом для продажи БАДов

## Структура проекта:

### 1. Типы и интерфейсы (src/types/index.ts)
- Типы для клиентов, БАДов, сообщений чата
- Интерфейсы для ИИ ответов

### 2. Данные (src/data/)
- clients.ts - демо-данные клиентов с профилями
- supplements.ts - каталог БАДов
- aiResponses.ts - шаблоны ответов ИИ

### 3. Компоненты (src/components/)
- ChatInterface.tsx - основной интерфейс чата
- ClientSelector.tsx - выбор клиента для демо
- MessageBubble.tsx - пузырь сообщения
- TypingIndicator.tsx - индикатор печати
- ClientProfile.tsx - профиль клиента
- SupplementCard.tsx - карточка БАДа

### 4. Хуки (src/hooks/)
- useChat.ts - логика чата и ИИ ответов
- useTypingEffect.ts - эффект печати

### 5. Утилиты (src/utils/)
- aiAgent.ts - логика ИИ-агента
- localStorage.ts - работа с локальным хранилищем

### 6. Главная страница (src/pages/Index.tsx)
- Основной интерфейс приложения

## Особенности:
- Реалистичные демо-данные (10 клиентов)
- Персонализированные ответы на основе данных клиента
- Каталог из 15+ БАДов
- Эффекты печати и анимации
- Адаптивный дизайн
- Темная/светлая тема
